# 效率网络EffNet (MobileNet的进步版本) 

EffNet是MobileNet和ShuffleNet的改进版本. EffNet的主要目标是减少计算量, 采用的技术有卷积分解和Depthwise separable convolution. 和MobileNet的用途一样, 主要希望将这种网络用于对手机等移动设备.
